# Snaze :snake:

## Como executar?
> - Clone este projeto.
> - Abra a pasta no seu terminal.
> - Execute o comando "mkdir build".
> - Execute o comando "cd build".
> - Execute o comando "cmake --build ."
> - Execute o comando "cd ../".
> - Execute o comando "./build/snaze ./data/maze1.txt"
